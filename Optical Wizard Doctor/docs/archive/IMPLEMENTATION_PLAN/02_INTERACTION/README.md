# Phase 2: Interaction

## Objective

Enable user interaction with the document: create regions, select them, move them, resize them, and navigate the document.

## Sub-Projects

1. **Canvas** - PDF display and region rendering
2. **Region Operations** - Create, select, move, resize regions
3. **Navigation** - Zoom, pan, scroll

## Success Criteria

- Users can load a PDF and see it displayed
- Users can create regions by clicking and dragging
- Users can select regions
- Users can move selected regions
- Users can resize regions using handles
- Users can zoom and pan the document

## Estimated Time

3-4 weeks
